# proyecto-aquaSenseCloud
El proyecto AquaSenseCloud es una infraestructura automatizada en AWS para el análisis de datos de temperatura. Emplea S3 para la ingesta de CSVs , funciones Lambda para procesarlos y DynamoDB para almacenarlos. Incluye alertas SNS para anomalías y una API RESTful en Flask con Docker sobre EC2 para consultar las métricas
